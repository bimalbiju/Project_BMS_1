# Smart Bank CLI (Python)

## 👨‍💻 Developed By
Bimal Biju

## 📌 Project Description
Smart Bank CLI is a command-line based banking system developed using Python. 
It allows users to create accounts, deposit money, withdraw money, and check account balance.

## 🚀 Features
- Create Account
- Deposit Money
- Withdraw Money
- Check Balance
- Data stored using JSON file

## 🧠 Concepts Used
- Object-Oriented Programming (OOP)
- File Handling (JSON)
- Exception Handling
- Menu-driven program

## ▶️ How to Run
python bank.py
